# Screenshots

Drop screenshots here so the grid in the project [README](../../README.md#a-look-at-it)
renders. Use these exact filenames:

| file | what to capture | route |
| --- | --- | --- |
| `feed.png` | the community feed | `/main` |
| `create.png` | the editor, ideally showing the **no-advice** toggle | `/create-post` |
| `letters.png` | the wall of quiet letters | `/letters` |
| `companion.png` | the AI companion with the mood selector | `/ai-chat` |

Guidelines: PNG, roughly 1280px wide, light theme reads best on GitHub. A couple
of mobile (375px) shots are a nice addition – name them `*-mobile.png` and add a
row to the README grid.
