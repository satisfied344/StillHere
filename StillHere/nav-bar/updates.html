may 26, 2026 performance new fix

huge speed jump + you can finally edit your posts

the site got ~10× lighter on first load, the home page renders fast even on slow connections, and posts are now editable from the 3-dot menu — title, body, photos, all of it.

  • image assets: 28.6 MB → 2.7 MB — paper textures were 6900×10000 px for decorative use; aggressively resized + recompressed (−91%)
  • blocking <script> tags in <head> moved to defer; Google Fonts merged 3→1; preconnect hints; loading="lazy" on below-fold images
  • expected FCP in India: ~18s → ~2–3s · LCP: ~18s → ~3–5s · CLS on mobile: 0.38 → ~0.05 (the missing <meta viewport> was found)
  • edit your own posts — 3-dot menu on cards + the post page; loads the existing data into create-post, save → updates in place
  • dark-mode polish: paper-note "not alone" stays cream, doodles invert correctly (cats, clouds, asterisks), tag pills now high-contrast on both themes
  • native theme detection via prefers-color-scheme — site auto-picks dark / light from the OS on first visit
  • post-menu in Russian shortened so every item fits one line; the Edit / Delete divider moved to its proper place
may 23, 2026 design fix

mobile-friendly across the board

dedicated mobile.css shipped — small-screen layouts for every page, no more horizontal scroll on phones, navbar and footers properly reflow.

  • new mobile.css stylesheet linked from every page
  • dark-theme spot fixes (filter panel, refresh button, sidebar pills)
  • small UI tweaks across about / contacts / updates
may 21, 2026 design fix

warmer dark, stronger cards, language toggle visible

another pass on the dark palette — cards now stand noticeably above the background, secondary text is lighter, borders more visible. the language toggle in preferences actually shows up in dark mode now.

  • dark-theme.css reworked — card elevation, text contrast, border opacity
  • i18n toggle visible + working in dark mode
  • changelog and about-page copy edited
may 20, 2026 design fix

dark mode fixes across every page

fixed theme-breaking inline colors and textures left over from the light design, and rebuilt the live-presence counter so "online right now" reflects actual per-tab sessions, not stale counts.

  • inline color overrides removed — dark theme now applies cleanly everywhere
  • session-aware site_pings — stable per-tab id, no duplicates
  • several hardcoded cream/white values replaced with CSS tokens
may 19, 2026 new community

two languages, live presence, privacy-only analytics

client-side i18n in english & russian, a quiet on-site visitor counter, and analytics that don't track you. nothing about you, only that the page was opened.

  • JS/i18n.js — full EN/RU dictionary + toggle in preferences
  • site_pings — supabase table: timestamps + session-id only; powers the "x online now" counter on /statistics
  • Vercel Speed Insights + Web Analytics — no cookies, no profiling
  • shared supabase client — silences duplicate GoTrueClient warnings
  • startup-page.html renamed to index.html, vercel.json simplified
may 18, 2026 design security

warm dark mode + the moderation backbone

  • dark-theme.css — completely rewritten, warm-neutral tokens
  • docs.css — guidelines / privacy / terms / crisis pages styled consistently
  • SQL migrations: reports, admin_roles, moderation_log, user_blocks
  • strict-review supabase edge function — re-checks flagged content with stricter AI prompt
  • admin queue page: keep / shadow / remove + block-author duration picker
